Hola! I mi nombre es Mai. fui creada como un Bot multiproposito para hacer mas amena tu estadia en los servidores de Discord, tengo un sistema de Perfiles propio, Alertas de Streaming en Twitch, Frases Aleatorias de Anime y mas...

Invitame!

Principales Caracteristicas

Sistema de Perfiles

Mi sistema de perfiles te permite tener un perfil y personalizarlo a tu gusto con un Banner, biografia personalizados y un Titulo que podras comprar en nuestra tienda, Nuestra Moneda "MaiCoins" puedes conseguirla Diariamente con ¡daily

Frases Anime

Cuento con una base de datos de mas de 10.000 Frases Animes las cuales puedes mostrar aleatoriamente con ¡Quote

Configuracion Canales/Prefijo

Cuento con una lista de comandos para configurar donde pueden escribir mis comandos y donde anunciare a los Streamers de Twitch o el prefijo para usarme

Configuracion

Modulo de Configuracion de Mai

¡prefix <prefijo>

Cambia <prefijo> por el nuevo prefijo para el bot

¡setchannel

Solo podras usar el Bot en el canal que ejecutes este comando

¡resetchannel

Resetea la configuracion de $setchannel para configurar un nuevo canal para el Bot

Twitch

Modulo de Twitch

¡twitch setbot

Establece el canal de chat en donde anunciare cuando un Streamer este Online

¡twitch <usuario> <*mensaje>

Remplaza <usuario> por un usuario de twitch y  <*mensaje> por el mensaje que quieres mostrar cuando haga stream     PD: el mensaje puede contener espacios

Perfil

Modulo de Perfil

¡perfil

Muestra tu perfil

¡perfil @usuario

Muestra el perfil del usuario que menciones

¡perfil settitulo

Muestra todos los titulos que compraste o hayas adquirido en eventos especiales

¡perfil settitulo <id>

Remplaza <id> por un id de titulo que poseas para establecer

¡perfil setbiografia <biografia>

Remplaza <biografia> por tu nueva descripcion para mostrar

¡perfil setbanner <url>

Remplaza <url> por un enlace para tu nuevo banner solo se admite (jpg,png,gif)

Utilidad

Modulo de Utilidad

¡emoji <emoji>

Remplaza <emoji> por un emoji y obtendras la url hacia su imagen para que puedas usarla como desees

¡quote

Obtienes una frase aleatoria de Anime con su respecto autor y titulo
